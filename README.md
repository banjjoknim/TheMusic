# TheMusic
개인 창작자를 위한 음원 공유 웹 사이트

## 1. 기술스택
- Tools : `ECLIPSE IDE, SQL DEVLELOPER`
- Language : `Java, JavaScript, HTML5, CSS3`
- Database : `Oracle`

## 2. 기능
- 음원 관련
  - 음원 조회 및 등록
  - 음원 재생 및 정지
  - 좋아요(미구현)
  - 플레이리스트(미구현)
- 앨범 관련
  - 앨범 조회 및 등록
  - 좋아요(미구현)
  - 플레이리스트(미구현)
- 유저 관련
  - 로그인
  - 로그아웃
  - 회원가입
  - 유저 정보 조회 및 수정
- 검색
  - 제목별
  - 가수별
  - 앨범별
